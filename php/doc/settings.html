Changing the settings

Click the settings icon in the toolbar to bring up the settings window.

Most of the settings are experimental or for debug, but two of the entries may be useful:

  • Display period - change this setting to show more (or less) hours in the listings page. Obviously the more hours you show, the less detail will be visible without clicking the program.
  • Popup prog info - I originally wrote the interface so that hovering over a program popped up the program details window. After using the interface for a while, I got annoyed with the window popping up all the time so I changed the interface to require a click before the window is shown. If you set this flag, the interface will revert to the "pop up on hover" mode.

Note that all settings are stored in cookies. So changing the settings will only persist on that PC (and that web browser). Note also that the PS3 web browser does not keep cookies between power cycles.
